# AI Rules

- Không được tự ý thay đổi folder structure.
- Docs là source of truth.
- Không được tự tạo variant ngoài docs.
- Không được hardcode màu.
- Không được bỏ qua React Aria.
- Không được dùng HTML primitive nếu docs yêu cầu React Aria primitive.
- Luôn tạo type trước khi tạo component.
- Luôn tạo skeleton trước khi implementation.
